# IEEE MANIT Student Branch Recruitment Portal

> 🚀 **Project Status:** Complete / First Web Development Project 

A modern, responsive web application designed specifically for the **IEEE Student Branch (MSB) Recruitment Drive at MANIT Bhopal**. This portal functions as a dedicated induction landing page to showcase the committee's culture, core flagships, and society values to prospective members.

> ⚠️ **Disclaimer:** This is **not** the official institutional website for IEEE MSB. It is a custom-built recruitment portal designed to guide applicants through the drive and introduce them to the branch's active domains.

---

## 📌 Project Overview

This website serves as my **very first web development project**, built to combine modern front-end design choices with a functional user experience. The site aims to captivate students exploring tech communities by offering deep insight into what joining IEEE MSB entails—ranging from tech innovation to societal impacts.

### Key Highlights
* **Thematic Branding:** Styled with a dark, high-contrast digital look using sleek glassmorphism and modern glowing color transitions (`#00ffcc` aqua highlights).
* **Chapter Insight:** Provides details on core initiatives like **AARAMBH** (social outreach), **SAMWAD** (Women in Engineering initiative), and **SCEECS** (international research conference).
* **Optimized Image Containers:** Structured layout models using `object-fit: cover` parameters to cleanly uniformize varying media aspect ratios.

---

## ✨ Features

* **Glassmorphism Navigation:** A fixed-top navigation panel built with a subtle background blur (`backdrop-filter`) to stay accessible during scrolling.
* **Interactive Hover Dynamics:** Smooth transitions and multiple neon text-glow shadow layers when interacting with the main menu.
* **Animated Elements:** Subtle entrance behaviors powered by native CSS `@keyframes fadeIn` engines to improve page landing feel.
* **Flexible Grid Layouts:** Clean display grids housing event images and informative content blocks that shift flexibly via CSS Flexbox.

---

## 🛠️ Tech Used & Concepts Applied

* **Markup Layer:** Semantic HTML5 layout tags (`<header>`, `<nav>`, `<section>`, `<footer>`).
* **Presentation Layer:** CSS3 properties including `flex-wrap`, custom keyframe triggers, layer transformations, and absolute/relative positioning frameworks.
* **Design Pattern:** Fluid layouts aiming for clean, screen-wide visibility across desktop setups.
